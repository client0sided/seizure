# Seizure UI

Seizure UI is a modern, optimized Roblox UI library inspired by WindUI.  
Create **Windows, Tabs, Buttons, Toggles, Sliders, and more** with minimal code.

![UI Screenshot](assets/seizure_ui_screenshot.png)

---

## Features

- Clean and modern dark theme
- Sidebar + tab navigation
- Reusable components
- Fully Roblox Studio compatible
- Optimized for performance
